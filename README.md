# BigOnlineShopping <br>
## 技术选型 <br>
>Maven（管理依赖及工程构建）

>Git（版本控制工具）

>SSM框架（spring+springmvc+mybatis）

>Mysql（数据库）,MyCat(数据库中间件)

>JSP、jQuery、jQuery EasyUI、KindEditor（富文本编辑器）

>Dubbo+zookeeper（调用&发布服务）

>Nginx（http&反向代理服务器）

>FastDFS（图片服务器）

>Redis（缓存服务器）

>Solr（搜索服务器）

>Freemarker（页面静态化）

>Activemq(消息队列)
